About
Description

A Classic Italic Roman with a set of alternative swash caps and a number of original swash lower case charactures that can create a number of unusual ligatures when connecting to the preceding characters. Available in two weights Light and Bold.

Type Designer
Alan Meeks

Alan Meeks

The ALAN MEEKS.COM typeface foundry was founded in 2008, since then it has been responsible for 25 faces including all genres of design sans, roman, script, decorative etc. and have sold extensively throughout the world. I work alone so all of the design, production and font making is done by me with the exception of Independence Script, which was a collaboration with top British calligrapher Satwinder Sehmi. Each font is based on the standard 250 character set with occasional alternatives and, often, non ranging numerals. All fonts are available from most font manufacturers and myself.

    How to Use

    You may encounter slight variations in the name of this font, depending on where you use it. Heres what to look for.

    Desktop

    In application font menus, this font will display:

    Park Lane Light

    Web

    To use this font on your website, use the following CSS: 

    
font-family: park-lane, serif;
font-style: italicnormal;
font-weight: 300;
